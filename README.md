## Project Structure

Each folder in this repository corresponds to a specific module or assignment:
(The M is for Module, T is for Topic. So M1T2 would be Module 1, Topic 2)

- M0T4 Hello World – Introductory console app
- M1T2 First Responsive Web App – Basic web layout with Razor
- M2T2 Movies List – Basic Crud setup. Also a demonstrating break point still present.
- M2T3 Phone Contacts - CRUD with data validation.
- M5T5 Menu Admin Routing - From here on, each project is added to the basis of this page. Different routing demonstrations.
- M6T4 WebSite Bootstrap - Demonstration of using bootsrap from dropdown menu and validation (albeit a very not secure validation).
- M7T1 Olympic Data – Tooltip-enhanced flag grid with dual filtering.
